# 导论　极限思想的起源# 导论：极限思想的起源



数学分析的真正开端，不是导数公式，也不是积分技巧，而是一个更朴素、也更深刻的问题：当一个量不断变化，并且"越来越接近"某个确定结果时，我们究竟在说什么？> 这是第一卷的导论。请在此撰写你的内容。


在初等数学中，人们习惯于处理有限对象。一个分数可以直接写出，一个方程可以直接求解，一个图形的面积也常常有现成公式。但一旦进入变化、逼近与连续现象，有限方法就显得不够用了。譬如，圆的面积怎样得到？曲线的切线斜率怎样定义？一个变动过程在"趋于某一点"时究竟呈现何种规律？这些问题有一个共同特征：答案并不是一步算出来的，而是在不断细分、不断逼近、不断控制误差的过程中显现出来的。极限思想，正是对这类问题的数学概括。

从表面看，极限只是"越来越近"；但从分析学的立场看，极限的本质并不是模糊的接近感，而是可以精确控制的逼近过程。换句话说，极限不是一种玄学式的"差不多"，而是一套关于误差如何变小、如何小到任意给定程度以下的严格语言。分析学之所以成为一门独立而深刻的学科，就在于它把这种原本依赖直觉的思想，改造为可以推演、可以证明、可以反复使用的理论工具。

## 一、极限思想的最早背景：从测量到穷竭

极限思想并不是近代数学家凭空发明出来的，它最早诞生于几何与测量的困难之中。古希腊数学已经非常重视证明，但他们处理的大多是有限图形与确定关系。一旦碰到圆、抛物线、球体等"非直线性"的对象，传统的分割与拼接方法就开始失效。

以圆面积为例。对于矩形，面积容易由长乘宽得到；对于三角形，面积也可以转化为底与高的乘积；但圆并不能直接分解成有限个标准图形。于是人们想到一种思路：既然圆本身难以直接计算，不妨先在圆内作正多边形，再在圆外作正多边形，用内接与外切的方法把圆"夹"在中间。随着边数不断增加，内接多边形的面积越来越大，外切多边形的面积越来越小，而两者之间的差距越来越小，圆的面积就被逐渐"逼"了出来。

这种方法就是著名的**穷竭法**。它的思想并不复杂，但意义极大。它说明，对于一个暂时无法直接把握的对象，我们未必要立刻"得到它"，而可以先构造一列越来越接近它的对象，并证明误差能够无限缩小。这个思想已经非常接近现代极限理论。

设半径为 $r$ 的圆内接正 $n$ 边形面积为 $S_n$，外切正 $n$ 边形面积为 $T_n$。则有

$$S_n \leq S \leq T_n,$$

其中 $S$ 表示圆的面积。随着 $n\to\infty$，若能证明

$$T_n - S_n \to 0,$$

那么圆面积 $S$ 就被唯一确定下来。这里最重要的不是具体算出某个数值，而是认识到：一个对象可以通过"双边夹逼"的无穷过程被精确把握。

不妨把这个过程写得再具体一些。正 $n$ 边形内接于半径为 $r$ 的圆时，其面积为

$$S_n = \frac{1}{2}nr^2\sin\frac{2\pi}{n}.$$

当 $n$ 很大时，$\sin\frac{2\pi}{n}\approx \frac{2\pi}{n}$，于是

$$S_n \approx \frac{1}{2}nr^2\cdot\frac{2\pi}{n} = \pi r^2.$$

这不是严格的证明，但它给了我们一种直觉上的踏实感：多边形确实在朝 $\pi r^2$ 靠拢。严格化这种"踏实感"，正是分析学要做的事。

穷竭法的成功不限于圆。阿基米德用类似的思路计算了抛物线弓形的面积。他把弓形不断分割成三角形，每一步填入的三角形面积恰好是上一步的 $\frac{1}{4}$，于是总面积变成了一个几何级数：

$$S = \Delta + \frac{\Delta}{4} + \frac{\Delta}{16} + \cdots = \frac{4}{3}\Delta,$$

其中 $\Delta$ 是最大三角形的面积。这个结论在两千多年前就已经被严格证明了。它告诉我们，无穷求和并不一定是不可控的——只要每一步加入的量衰减得足够快，总和就是有限的。这一点在后面讨论级数理论时还会反复出现。

这类思想后来在分析学中反复出现。一个函数的极限常常不是直接算出的，而是通过若干更简单的量加以逼近；一个积分值也不是突然"出现"的，而是在分割越来越细时逐渐稳定下来。可以说，穷竭法提供了分析学最原始的工作范式：**不直接抓住对象，而是控制逼近过程**。

## 二、芝诺悖论与"无穷过程"的危机

如果说穷竭法展示了无穷逼近的建设性一面，那么芝诺悖论则揭示了无穷思想的危险一面。它提醒人们：只要涉及"无限次""无限小""无限分割"，直觉很容易失效。

最著名的例子是"阿基里斯追龟"。阿基里斯要追上一只先出发的乌龟。他先到达乌龟原来的位置，但在此期间乌龟又向前爬了一段；他再到达这个新位置时，乌龟又前进了一点；如此下去，似乎阿基里斯永远都只能到达乌龟"刚刚离开"的地方，于是永远追不上。

这个论证的问题，不在于它列举了无限多个阶段，而在于它把"要经过无限多个步骤"误解成"永远无法完成"。现代数学告诉我们，关键不在步骤个数有限还是无限，而在于这些步骤对应的总时间是否收敛。

设乌龟初始领先距离为 $a>0$，阿基里斯速度为 $v_1$，乌龟速度为 $v_2$，并且 $v_1>v_2$。则阿基里斯追上乌龟所需时间应为

$$t = \frac{a}{v_1 - v_2}.$$

另一方面，按照芝诺式分段，这段时间可以写成一个无穷级数：

$$\frac{a}{v_1} + \frac{av_2}{v_1^2} + \frac{av_2^2}{v_1^3} + \cdots = \frac{a}{v_1}\left(1 + \frac{v_2}{v_1} + \left(\frac{v_2}{v_1}\right)^2 + \cdots\right).$$

由于 $\left|\frac{v_2}{v_1}\right| < 1$，故几何级数收敛，从而

$$\frac{a}{v_1}\cdot\frac{1}{1 - \frac{v_2}{v_1}} = \frac{a}{v_1 - v_2}.$$

这正是追上所需的有限时间。

不妨把数字代进去看一下。假设乌龟领先 $100$ 米，阿基里斯每秒跑 $10$ 米，乌龟每秒爬 $1$ 米。按芝诺式分段：第一阶段，阿基里斯跑 $100$ 米，用时 $10$ 秒，此时乌龟又爬了 $10$ 米；第二阶段，他跑 $10$ 米，用时 $1$ 秒，乌龟又爬了 $1$ 米；第三阶段，跑 $1$ 米，用时 $0.1$ 秒，乌龟爬了 $0.1$ 米……总时间是

$$10 + 1 + 0.1 + 0.01 + \cdots = \frac{10}{1 - 0.1} = \frac{100}{9} \approx 11.11 \text{ 秒}$$

这与直接计算 $t = \frac{100}{10 - 1} = \frac{100}{9}$ 完全一致。无穷多步加起来，总时间不过 $11$ 秒出头。

芝诺悖论的重要性，不在于它"难倒了数学"，而在于它逼迫数学家正视一个事实：无穷过程不能靠生活直觉处理，必须建立新的逻辑规则。无限多个量加起来可以是有限的；无限逼近并不意味着永远达不到；"可以无限分割"与"无法完成整体过程"并不是同一回事。分析学后来对极限、级数、连续的严格处理，本质上都在回答芝诺提出的挑战。

还有一个类似的例子值得一提。设你站在离墙壁 $1$ 米处，每一步走完剩余距离的一半：第一步走到 $\frac{1}{2}$ 米处，第二步到 $\frac{3}{4}$ 米处，第三步到 $\frac{7}{8}$ 米处……第 $n$ 步后你与墙壁的距离是 $\frac{1}{2^n}$ 米。直觉告诉你永远到不了墙壁，但数学告诉你：对于任意小的距离 $\varepsilon$，只要走够 $n > \log_2(1/\varepsilon)$ 步，你和墙壁的距离就小于 $\varepsilon$。你的位置序列 $1 - \frac{1}{2^n}$ 收敛到 $1$。"到不了"和"无限接近"，在数学中是两件不同的事。

我个人更愿意把芝诺悖论理解为分析学的哲学起点。它不是在否定运动，而是在问：如果世界包含连续变化，那么数学怎样才能描述这种变化而不陷入矛盾？这个问题一直延续到近代微积分的建立。

## 三、从几何问题到运动问题：极限思想的转向

古代的无穷方法主要服务于几何面积与体积问题，而到了近代，极限思想的重心逐渐转向运动与变化。这个转向极其关键。因为几何中的"穷竭"更多是在处理静态图形，而近代科学，特别是力学和天文学，开始要求数学去处理速度、加速度、轨迹、瞬时变化率等动态概念。

平均速度是容易定义的。若物体在时间区间 $[t, t+\Delta t]$ 内位移改变量为 $\Delta s$，则平均速度是

$$\frac{\Delta s}{\Delta t}.$$

但所谓"瞬时速度"，并不能直接通过一个长度为零的时间间隔来定义，因为当 $\Delta t = 0$ 时，上式失去意义。于是数学不得不采取一种新的策略：让 $\Delta t$ 不等于零，但不断趋于零，考察商

$$\frac{s(t + \Delta t) - s(t)}{\Delta t}$$

在 $\Delta t \to 0$ 时是否趋于某个确定值。这个确定值，就是瞬时速度，也就是导数。

例如，设

$$s(t) = t^2.$$

则

$$\frac{s(t+\Delta t) - s(t)}{\Delta t} = \frac{(t+\Delta t)^2 - t^2}{\Delta t} = \frac{2t\Delta t + (\Delta t)^2}{\Delta t} = 2t + \Delta t.$$

当 $\Delta t \to 0$ 时，

$$2t + \Delta t \to 2t.$$

因此 $s'(t) = 2t$。

这个例子看起来简单，但它标志着数学表达方式的一次根本变化。我们不是把"瞬时速度"当作一个神秘量直接规定出来，而是把它解释为平均速度在时间间隔无限缩小时的极限。这种处理方式后来推广为整个微分学的基本框架。

再看一个稍微不同的例子。假设一辆汽车的位移公式是 $s(t) = 4t - t^2$（单位：米和秒）。它在 $t = 1$ 秒时的瞬时速度是多少？

$$\frac{s(1+\Delta t) - s(1)}{\Delta t} = \frac{[4(1+\Delta t) - (1+\Delta t)^2] - [4\cdot 1 - 1^2]}{\Delta t} = \frac{(3 + 2\Delta t - (\Delta t)^2) - 3}{\Delta t} = 2 - \Delta t.$$

当 $\Delta t \to 0$，瞬时速度为 $2$ 米/秒。注意，这辆汽车的速度是在不断变化的——$s'(t) = 4 - 2t$，所以在 $t=2$ 秒时速度为零，之后开始倒退。导数不只是给出一个数值，它告诉我们运动的全部局部信息。

同样地，积分也并不是从"面积公式表"中长出来的，而是从"细分后求和"的思想中长出来的。若要计算曲边梯形面积，就把区间分成许多小段，在每个小段上取一个窄矩形，以矩形面积之和近似原图形面积。分割越细，近似越好；当分割长度趋于零时，这些和若趋于一个稳定值，这个值就被定义为定积分。

举一个最朴素的例子。想计算抛物线 $y = x^2$ 在 $[0,1]$ 上与 $x$ 轴围成的面积。把 $[0,1]$ 等分成 $n$ 份，每份宽度 $\frac{1}{n}$，在第 $k$ 个小区间的右端点取高 $\left(\frac{k}{n}\right)^2$，则矩形面积之和为

$$S_n = \sum_{k=1}^{n}\left(\frac{k}{n}\right)^2\cdot\frac{1}{n} = \frac{1}{n^3}\sum_{k=1}^{n}k^2 = \frac{1}{n^3}\cdot\frac{n(n+1)(2n+1)}{6} = \frac{(n+1)(2n+1)}{6n^2}.$$

当 $n\to\infty$ 时，

$$S_n \to \frac{2}{6} = \frac{1}{3}.$$

所以抛物线下方面积是 $\frac{1}{3}$。这个计算虽然初等，但它的思路——分割、求和、取极限——正是 Riemann 积分的原型。

所以，导数与积分看似处理不同问题，一个研究局部变化，一个研究整体累积，但它们在思想上是统一的：都依赖极限，都依赖"先近似、再逼近、最后取稳定结果"的工作方式。

## 四、牛顿与莱布尼茨：微积分的建立与方法论突破

17世纪是极限思想真正进入成熟阶段的时代。牛顿和莱布尼茨分别在不同背景下建立了微积分。牛顿的问题意识来自力学与天体运动，他关心的是变量随时间的"流动"；莱布尼茨则更强调符号系统与计算规则，他把微积分发展成一种高度可操作的算法语言。

从历史角度说，他们的伟大不只在于发明了若干公式，而在于第一次大规模地把无穷小、切线、面积、求和、变化率这些彼此分散的问题组织成一个统一理论。更重要的是，他们发现微分与积分互为逆运算，这使得局部变化与整体累积之间建立起深刻联系，也就是后来所谓的**微积分基本定理**。

用现代语言来说，如果 $f$ 是连续函数，$F(x) = \int_a^x f(t)\,dt$，那么 $F'(x) = f(x)$。反过来，

$$\int_a^b f(x)\,dx = F(b) - F(a).$$

这条定理的威力巨大。它意味着，如果你想计算一个面积（积分），你不必真的去做分割求和的极限过程——只要找到一个原函数就行。比如，$\int_0^1 x^2\,dx$ 不需要像刚才那样用 $n$ 等分去逼近，直接用 $F(x) = \frac{x^3}{3}$ 就得到 $F(1) - F(0) = \frac{1}{3}$。积分从一个极限问题变成了求原函数问题，计算效率提高了几个数量级。

但必须看到，早期微积分虽然极其成功，在逻辑上却并不完全稳固。牛顿讲"最终比"，莱布尼茨讲"无穷小量"，都带有强烈的直觉色彩。它们在实际计算中威力巨大，但在严格证明上仍留下了模糊地带。一个量既说它"不是零"，又说它"比任何已知量都小"；这样的说法在运算中方便，在逻辑上却并不令人满足。

爱尔兰主教贝克莱（Berkeley）曾经尖锐地批评过这种做法。他讽刺道："它们既不是有限量，也不是无穷小量，也不是虚无。我们可不可以把它们叫做逝去量的鬼魂？"这种批评虽然尖刻，却并非毫无道理——它逼迫数学家正视自己使用的概念究竟是否站得住脚。

这恰恰说明数学发展的一个常见规律：理论往往先在问题推动下形成有效方法，然后再由后人补上基础。微积分不是先有一套完美公理再去应用，而是先在科学问题中获得成功，再倒逼数学家去追问：我们到底在使用什么概念？这些概念能否被严格定义？极限由此从"技巧背景"上升为"理论核心"。

## 五、19世纪的转折：从直觉极限到严格极限

到了19世纪，随着函数概念的扩展和级数问题的深入，数学家越来越清楚地认识到，仅凭几何直观和物理背景已经不足以支撑分析学。尤其是在处理无穷级数、反常现象、处处连续但不可导函数等问题时，旧有直觉不断失灵。于是，极限必须被严格化。

举一个让当时数学家吃惊的例子。考虑 Fourier 在研究热传导时遇到的问题：一个处处连续的函数，能否用三角级数（正弦和余弦的无穷求和）来表示？如果能，这个级数在每一点都收敛吗？如果逐项对这个级数求导，结果还对吗？这些看似自然的操作，一旦缺乏严格的收敛性理论，答案可能出人意料地混乱。正是这些困难促使数学家发展一致收敛等更精细的概念。

还有一个更令人震惊的例子。Weierstrass 在 1872 年构造了一个**处处连续但处处不可导**的函数。人们原本以为，连续函数"大部分地方"总是光滑的，最多有几个不可导的"尖角"。Weierstrass 用一个显式的三角级数证明这种直觉完全错误。这类反例的出现，迫使数学家把"连续""可导""收敛"这些概念彻底从直觉中剥离出来，赋予它们不依赖图形想象的纯逻辑定义。

这一时期的核心工作，是把"无限接近"改写成关于误差的精确定义。现代数学中常见的 $\varepsilon$-$\delta$ 语言，就是在这种背景下形成的。它的根本精神可以概括为一句话：**一个量趋于某值，不是因为我们"感觉它越来越近"，而是因为无论你预先规定多么小的允许误差，我都能找到足够靠后的阶段，使得以后所有值都落在这个误差范围内。**

例如，数列 $a_n$ 收敛于 $A$，是指对于任意 $\varepsilon > 0$，存在正整数 $N$，使得当 $n > N$ 时，总有

$$|a_n - A| < \varepsilon.$$

这一定义的高明之处在于，它完全摆脱了"差不多""无限小"这类模糊说法，而把极限变成可以检验、可以证明、可以推广的数学对象。

举一个最简单的例子。设

$$a_n = \frac{1}{n}.$$

要说明 $a_n \to 0$，只需证明：任意给定 $\varepsilon > 0$，总可以找到 $N$，使得当 $n > N$ 时，

$$\left|\frac{1}{n} - 0\right| < \varepsilon.$$

只要取

$$N > \frac{1}{\varepsilon},$$

便可保证 $n > N \Rightarrow \frac{1}{n} < \varepsilon$。于是极限成立。

再看一个稍复杂的例子。设 $a_n = \frac{3n+1}{2n+5}$。直觉上，当 $n$ 很大时，$+1$ 和 $+5$ 无足轻重，商约为 $\frac{3}{2}$。但"约为"在数学中不算数。严格地说，要证明 $a_n \to \frac{3}{2}$，就要对任意 $\varepsilon > 0$ 找到 $N$，使 $n>N$ 时

$$\left|\frac{3n+1}{2n+5} - \frac{3}{2}\right| < \varepsilon.$$

化简：

$$\left|\frac{2(3n+1) - 3(2n+5)}{2(2n+5)}\right| = \left|\frac{-13}{2(2n+5)}\right| = \frac{13}{2(2n+5)}.$$

要让它小于 $\varepsilon$，只需 $2n+5 > \frac{13}{2\varepsilon}$，即 $n > \frac{13 - 10\varepsilon}{4\varepsilon}$。取 $N$ 为大于此值的任意正整数即可。

这种论证第一次做时会觉得繁琐，但它训练的恰恰是分析学最核心的能力：**把"感觉上对"变成"证明上对"**。写出这样一段 $\varepsilon$-$N$ 论证，就像给一个直觉配上了一副可以受力的骨架。

这个例子虽然初等，却揭示了分析学的真正气质：它不是依赖灵感的猜测，而是依赖对误差的精确控制。极限一旦被如此定义，连续、导数、积分、级数一致收敛等概念也都获得了可靠基础。

## 六、为什么分析学必须从极限开始

很多初学者会有一个疑问：为什么教材一上来不先讲求导公式、不先讲积分技巧，而偏偏要从实数、数列、极限这些看似抽象的内容讲起？原因很简单，因为后面的全部结论都建立在极限之上。如果极限不清楚，导数只会沦为机械公式，积分只会变成算题程序，学生很难真正理解分析学在做什么。

不妨举一个具体的例子来说明"不理解极限"可能导致的问题。考虑级数

$$1 - 1 + 1 - 1 + 1 - 1 + \cdots$$

如果"随意求和"，我们可以得到至少两种"答案"：
- 把相邻两项配对：$(1-1) + (1-1) + \cdots = 0 + 0 + \cdots = 0$
- 先取第一项，再把后面配对：$1 + (-1+1) + (-1+1) + \cdots = 1 + 0 + 0 + \cdots = 1$

同一个式子给出 $0$ 和 $1$ 两个答案？甚至 Euler 曾经提出这个级数的和"应该"是 $\frac{1}{2}$。哪个对？事实上，这个级数**发散**——它的部分和交替为 $1$ 和 $0$，永远不稳定在一个值上。没有严格的极限定义，这类问题会永远争论不休。

从知识结构看，数学分析的主要对象是"变化中的量"和"由无限过程得到的结果"。凡是涉及以下情形，就离不开极限：

- 当自变量逐渐靠近某点时，函数值是否趋于稳定；
- 当分割无限变细时，面积和是否趋于某个定值；
- 当项数无限增多时，和式是否仍有意义；
- 当逼近过程不断继续时，误差能否被任意压小。

这些问题虽然形式不同，但核心结构完全一致。它们都在研究"一个过程是否趋向某个确定对象"。因此，极限并不是分析学中的一个孤立章节，而是整门课程的共通语言。可以说，谁真正理解了极限，谁才算真正进入了分析学。

在教学上，我认为极限最难的地方，不是定义本身，而是**思维方式的转变**。中学数学偏重"求值"，而分析学更关心"过程"；中学数学常问"结果是多少"，而分析学常问"为什么这个过程一定稳定到这个结果"。这两种问法看似接近，其实训练的是不同的智力结构。前者偏计算，后者偏控制。分析学的成熟，恰恰体现在它学会了控制无穷过程。

## 七、本导论在全书中的位置

本节不是单纯的数学史，也不是为后文做装饰性的开场。它承担三个任务。

**第一**，它说明极限思想并非人为制造的技术，而是由几何、运动、测量、科学计算等一系列根本问题逼出来的。也就是说，极限不是数学家的偏好，而是连续世界对数学提出的要求。

**第二**，它提示学生：无穷过程既强大又危险。只要处理不当，就会出现悖论、误判和逻辑漏洞。所以分析学后面的严格定义，不是故作艰深，而是对这种危险的必要回应。

**第三**，它给全书定下方法论基调：分析学不是研究"孤立的数"和"孤立的式子"，而是研究对象在变化、逼近、分割、叠加过程中的稳定行为。数列极限、函数极限、连续性、微分学、积分学、级数理论，表面上分属不同章节，实则都在回答同一个问题：**一个无限过程能否导向一个确定结果？**

从这个意义上说，极限思想不是数学分析的开头之一，而是数学分析真正的精神主轴。

## 小结

极限思想起源于人类对连续对象的把握困难。古希腊的穷竭法表明，复杂对象可以通过不断逼近而被刻画；芝诺悖论则说明，无穷过程若缺乏严格规则，直觉就会陷入混乱。到了近代，牛顿与莱布尼茨把这种思想系统化为微积分；到了19世纪，数学家进一步把"接近"改写为可精确控制的误差语言，极限终于获得严格基础。此后，分析学的一切主要概念都在极限框架中展开。

因此，学习数学分析，不能把极限看成一个必须硬啃的前置定义，而应把它看成理解整门课程的入口。极限并不神秘，它研究的只是这样一件事：**当一个过程无限推进时，数学如何保证我们仍然在谈论一个确定、可靠、可证明的结果。**
